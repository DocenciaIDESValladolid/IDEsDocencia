Prototipo 1

Más información Mapa Eléctrico
  1. Uso de tecnologías IDE - La aplicación utiliza datos y servicios distribuidos enlazados mediante recomendaciones OGC.
  2. Base de datos espaciales  - Gestiona la información en una base de datos PostgreSQL/PostGIS.
  3. Nodo IDE - Servidor Geoserver configurado como nodo IDE con la información propia de la aplicación.
  4. Applicación móvil - Aplicación de cliente ligero con Javascript para implementar el prototipo de un servicio geo-referenciado.
