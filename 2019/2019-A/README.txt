Este proyecto contiene tres directorios principales:

- app: contiene todos los ficheros necesarios para la parte cliente de la aplicación. En cuanto
	a la "lógica de aplicación" (implementación de las funcionalidades), esta se encuentra en
	el fichero app/src/app.js. También se han modificado los ficheros app/app.html y app/index.html
	para adaptarlos a nuestra aplicación.
- database: contiene los ficheros e información necesaria para las bases de datos
- docs: ficheros con explicación de la configuración delas bases de datos (docs/Database.docx) y para
	la configuración del servidor GeoServer (Configuracion_GeoServer.docx)
